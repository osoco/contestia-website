Te ayudamos a elegir sólo lo que realmente merece la pena

Contestia ofrece una versátil plataforma de selección para diferentes usos, bien sea para un concurso, selección de recursos humanos, valoración de eventos, tenemos la solución.

  • Concursos

    Diseña y gestiona todos los pasos de tu concurso. Accede a la información de tus participantes. Todo desde la misma herramienta.

  • Recursos Humanos

    Crea tu proceso de selección, personaliza las diferentes fases y gestiona a tus candidatos de forma sencilla.

  • Eventos

    Selecciona los mejores ponentes para tu evento de foma sencilla, simplificando el proceso de Call for papers.

  • Otros usos

    Si tienes necesidad de organizar un proceso de selección de otro tipo, habla con nosotros, podemos proporcionarte una solución a medida.

¿Cómo funciona?

  1. Widget integrable

    Ofrecemos un widget personalizado para recoger los datos necesarios en cada caso para su posterior gestión y evaluación. Este widget se puede integrar en vuestra web, la web de otra plataforma, o incluso podemos alojarlo nosotros.

  2. Gestión de flujos

    Diseña los procesos y las fases por las que pasará cada solicitud. Asigna revisores para cada fase de manera que el resultado sea el óptimo en tu caso.

  3. Filtros y etiquetado

    Podrás establecer filtros para que los candidatos se muevan de una ronda a la siguiente atendiendo a un criterio específico. Para esto además, podrás utilizar etiquetado específico para tu proyecto. Nuestros filtros son muy avanzados y permiten sumar condiciones para ser muy precisos.

  4. Herramientas de evaluación

    Con nuestra herramienta de evaluación, exclusiva para uso de los revisores/evaluadores, podrán revisar fácilmente cada candidato, evaluar y corregir sus evauaciones hasta tener unas respuestas claras que enviar al proceso.

  5. Quédate con lo bueno

    Finalizado el proceso de filtrado, tras pasar por las fases que necesites, tendrás un resultado de candidatos depurado de manera sencilla. En la herramienta, además, dispondras de toda la información para hacer el seguimiento del flujo de cada candidato.
